# ? CodeNCash - Quick Start

## ?? Launch in 3 Commands

```bash
# 1. Install
pip install -r requirements-frontend.txt

# 2. Setup API Key
echo "GROQ_API_KEY=your_key_here" > .env

# 3. Run
cd frontend && streamlit run codencash_app.py
```

**Open:** http://localhost:8501
